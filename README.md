## 🧠 Projects

### 1. BiLSTM for Named Entity Recognition (Assignment 3)
- **Task:** SemEval-2021 MeasEval — extract scientific measurements 
  and quantities from research documents
- **Model:** Bidirectional LSTM (TensorFlow/Keras) with Embedding, 
  BiLSTM, and TimeDistributed Dense layers
- **Key skill:** Sequence labeling, token-level classification, 
  deep learning NLP

### 2. BERT Fine-tuning for Commonsense Validation (Assignment 4)
- **Task:** SemEval-2020 ComVE — classify which statement violates 
  commonsense reasoning across 3 subtasks
- **Model:** Pre-trained BERT fine-tuned via HuggingFace 
  AutoModelForSequenceClassification and Trainer API
- **Key skill:** Transformer fine-tuning, transfer learning, 
  modern LLM workflows
