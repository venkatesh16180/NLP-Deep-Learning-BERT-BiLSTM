# Assignment 3 — Sequence Labeling with BiLSTM
**Course: Applied Natural Language Processing | University of Arizona**

## 📌 Task Overview
Named Entity Recognition (NER) on the **SemEval-2021 MeasEval** shared task — 
extracting counts, measurements, and scientific quantities from research documents.

## 🧠 Model Architecture
Built a **Bidirectional LSTM (BiLSTM)** neural network using TensorFlow/Keras:
- Embedding layer for token representations
- Bidirectional LSTM layer to capture context from both directions
- TimeDistributed Dense output layer for per-token classification
- Reproducibility controls via seed initialization
